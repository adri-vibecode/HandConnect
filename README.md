# HandConnect
HandConnect: Advanced Hand Tracking AR
HandConnect is an immersive, browser-based Augmented Reality experience that leverages real-time hand tracking and generative visuals. Built using the MediaPipe Hands API and HTML5 Canvas, it detects hand gestures and transforms them into interactive, neon-infused light displays.
🚀 FeaturesReal-Time 
Hand Tracking: Powered by Google's MediaPipe, providing precise hand landmark detection with low latency.  
Interactive Visuals: Dynamic, physics-based particles and ripples that respond to your movements. 
Gesture Recognition: Detects pinches and hand states to trigger visual "zaps" and audio feedback.  
Thematic UI: Switch between multiple aesthetic themes, including:
🌈 Rainbow
🤖 Cyberpunk
🔥 Lava
🌊 Ocean
🌌 GalaxyProcedural
Audio: Includes a reactive audio engine that responds to the proximity of your hands.  
🛠️ Tech Stack:
HTML5 / CSS3: Used for the glassmorphism UI and layout.  
JavaScript (Vanilla): Logic for physics, gesture detection, and state management. 
MediaPipe Hands: The core machine learning library for hand tracking.  
Web Audio API: Used to create the reactive sonic environment.  
💻 Getting Started
Prerequisites
A modern web browser (Chrome, Edge, or Firefox).
A working webcam.
🎮 How to Use:
Grant Permissions: Upon opening the page, the application will request access to your camera to enable the tracking engine.  
Start: Click the "Enter Experience" button to initialize the audio context and start the AR engine.  
Interact:Move your hands in view of the camera to see the skeletal tracking and neon trails.  
         Bring your thumb and index finger together to perform a "Pinch" and trigger visual shockwaves.  
         Use the theme menu at the bottom of the screen to switch visual styles.
